# SQL Learning Journey: A to Z

## Topics Covered in This Complete Course

### Introduction
Learn what SQL is, why it matters, how databases work, and how to set up your full SQL environment.

### Querying Data (SELECT)
Master `SELECT`, `FROM`, `WHERE`, `GROUP BY`, `HAVING`, `ORDER BY`, `DISTINCT`, `TOP`, and query execution order.

### Data Definition (DDL)
Create, modify, and remove database objects using `CREATE`, `ALTER`, and `DROP` commands.

### Data Manipulation (DML)
Add, update, and delete records using `INSERT`, `UPDATE`, and `DELETE` with real-world logic.

### Filtering Data
Use comparison and logical operators like `AND`, `OR`, `NOT`, `BETWEEN`, `IN`, and `LIKE` to filter data effectively.

### Combining Data
Join and merge tables using `INNER`, `LEFT`, `RIGHT`, `FULL`, `CROSS` joins and set operations like `UNION`, `INTERSECT`.

### Row-Level Functions
Use string, numeric, date, null-handling functions, and `CASE` expressions to transform your data.

### Aggregation & Analytics
Apply aggregate functions and advanced window functions like `RANK`, `DENSE_RANK`, `LAG`, and `LEAD`.

### Advanced SQL Techniques
Work with subqueries, CTEs (recursive and non-recursive), views, temp tables, procedures, and triggers.

### Performance Optimization
Improve query speed using indexes, partitions, and practical performance tips.

### AI & SQL
Use ChatGPT and GitHub Copilot to generate, explain, optimize, and debug SQL — plus translate and document code.
